# Three Minute Demo Script

## Goal

Show judges that FunnelOps Autopilot is a real business workflow agent powered by Qwen Cloud, not a one-shot prompt demo.

## 0:00 to 0:20, Problem and Track

Say:

FunnelOps Autopilot is built for Track 4, Autopilot Agent. It helps solo sellers and small service businesses turn scattered inbound leads from DMs, forms, comments, and WhatsApp-style chats into memory-aware sales actions with human approval.

Show:

- App title.
- Track label.
- Inbound inbox with multiple lead sources.

## 0:20 to 0:55, Messy Lead Input

Show:

- Select the WhatsApp-style lead or Instagram DM lead.
- Point out the message contains pricing or setup questions.

Say:

This is the kind of lead small businesses often lose. It has buying intent, but it needs qualification, memory, and careful approval before any quote goes out.

## 0:55 to 1:35, Qwen Agent Run

Action:

- Click Run Agent.

Show:

- Intent.
- Urgency.
- Confidence.
- Draft reply.
- Next best action.
- Retrieved memories.

Say:

The backend calls Qwen Cloud server-side. Qwen classifies the lead, retrieves relevant business memories, drafts a response, and recommends the next action.

## 1:35 to 2:05, Human Approval and Trace

Show:

- Approval Queue.
- Agent Trace.
- Human approval required step.

Say:

Important actions do not go out automatically. Quotes, uncertain responses, and high-impact replies are held for the owner to approve, edit, or reject. The agent trace makes the decision path visible.

## 2:05 to 2:35, Memory and Usage Proof

Show:

- Memory Operations panel.
- Stale memory downgraded.
- Qwen Cloud Usage Ledger.

Say:

The memory layer scores memories by importance, confidence, freshness, and relevance. Stale memories are downgraded. The usage ledger shows Qwen runs and token usage so cloud usage is transparent.

## 2:35 to 3:00, AI Revenue Advisor and Submission Proof

Show:

- AI Revenue Advisor panel.
- Architecture diagram or README.

Say:

FunnelOps also includes an AI Revenue Advisor that analyzes the inbox, memories, workflow history, and token usage to recommend the next business move. The backend is designed for Alibaba Cloud deployment and Qwen Cloud powers the reasoning layer.

End with:

FunnelOps Autopilot is a lightweight, memory-rich inbound sales autopilot for people without a sales team.
